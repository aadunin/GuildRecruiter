# 📘 GuildRecruiter

**GuildRecruiter** — аддон для *World of Warcraft Retail*, созданный для удобного и эффективного рекрутинга в гильдию.  
Он позволяет отправлять заранее подготовленные сообщения, использовать шаблоны, рандомизацию, автоматически подставлять ссылку на гильдию и выводить служебные сообщения в отдельную вкладку чата.

---

## ✨ Возможности

- **Отправка рекрутинговых сообщений** в любой канал:  
  `SAY`, `YELL`, `GUILD`, `PARTY`, `RAID`, `CHANNEL`.
- **Шаблоны сообщений**: добавление, редактирование, удаление, просмотр списка.
- **Рандомизация** с поддержкой весов и «скользящего окна» для предотвращения повторов.
- **Автоматическая подстановка ссылки на гильдию** через режим `/gru share`.
- **Служебная вкладка чата** для вывода системных сообщений аддона.
- **Автопереключение чата** на выбранный канал перед отправкой.
- **Удобная slash‑команда `/gru`** с большим набором подкоманд.
- **Сохранение настроек** между сессиями (SavedVariables).

---

## 📦 Установка

1. Скачайте ZIP‑архив из раздела **Releases**.
2. Распакуйте в:
   ```
   World of Warcraft/_retail_/Interface/AddOns/GuildRecruiter
   ```
3. Запустите игру и убедитесь, что аддон включён.

---

## 🖥️ Использование

Основная команда:

```
/gru
```

### Основные подкоманды

| Команда | Описание |
|--------|----------|
| `/gru msg <текст>` | Установить основное сообщение |
| `/gru chan <TYPE> [id]` | Выбрать канал отправки |
| `/gru send` | Отправить сообщение |
| `/gru random on/off` | Включить/выключить рандомизацию |
| `/gru addtmpl <текст>` | Добавить шаблон |
| `/gru listtmpl` | Показать шаблоны |
| `/gru deltmpl <номер>` | Удалить шаблон |
| `/gru edittmpl <номер> <текст>` | Изменить шаблон |
| `/gru window <число>` | Размер «скользящего окна» |
| `/gru sysframe <name|index|off>` | Настроить вкладку для служебных сообщений |
| `/gru listchannels` | Список доступных каналов |
| `/gru status` | Показать текущие настройки |
| `/gru share` | Режим автосборки сообщения с ссылкой на гильдию |
| `/gru help` | Справка |

---

## 🔗 Режим автосборки сообщения (`/gru share`)

1. Введите `/gru share`.
2. Откроется окно гильдии.
3. Нажмите **«Пригласить участника» → «Поделиться в чате»**.
4. Аддон автоматически:
   - подставит ссылку на гильдию,
   - добавит ваше сообщение или шаблон,
   - переключит чат на нужный канал.
